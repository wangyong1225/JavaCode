ch02-di-xml

di:依赖注入，表示创建对象，给属性赋值

di的实现语法有两种：
1.在Spring的配置文件中，使用标签和属性完成，叫做基于xml的低实现
2.使用Spring中的注解，完成属性赋值，叫做基于注解的id实现

di的语法分类：
1、set注入（设置注入）：Spring调用类的set方法，在set方法可以实现属性的赋值

2、构造注入，Spring调用类的有参构造方法，创建对象。在构造方法中完成赋值
